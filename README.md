# hello_ucas
国科大课件同步、查看作业信息



## 1.使用方法

1. 将`hello_ucas.py`放到需要下载课件的根目录下

2. 添加`config.txt`（注意与`hello_ucas.exe`同级），填写登录信息，格式如下（一共两行）：

   ```
   username: XXXXXXXXX
   password: XXXXXXXXX
   ```

3. 运行`hello_ucas.py`，根据提示操作
